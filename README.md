# ICC Cricket World Cup 2019 – Data Analysis Project

A **Exploratory Data Analysis (EDA) project** implemented in **Python**, focusing on match statistics and venue-based insights

---

## Overview
This project implements an **Exploratory Data Analysis (EDA)** on the ICC Cricket World Cup 2019 dataset.  
The analysis focuses on understanding match outcomes, total matches played, abandoned matches, and stadium-wise match distribution.  
The project follows a structured EDA workflow to extract meaningful insights using Python data analysis libraries.

---

## Core Features
- Dataset loading 
- Data understanding and preprocessing  
- Analysis of total matches played  
- Identification of matches with no result or abandonment  
- Stadium-wise match hosting analysis  
- Data visualization using charts and plots  
- Extraction of key insights from the analysis  

---

## Technology Stack
- **Language:** Python  
- **Libraries Used:** NumPy, Pandas, Matplotlib, Seaborn  
- **Execution Environment:** Jupyter Notebook  

---

## Application Mechanics
- Match data is stored and processed using **Pandas DataFrames**  
- Exploratory analysis is performed using filtering, grouping, and aggregation techniques  
- Visualizations are generated to support analytical findings  
- The analysis answers the following questions:
  - Total number of matches played in the dataset  
  - Number of matches that ended without a result  
  - Stadium that hosted the most matches  
- Final observations are summarized as key insights  

---

- **icc-cricket-world-cup-2019-eda/**  
  - `ICC_Cricket_World_Cup_2019_EDA.ipynb` → Jupyter Notebook containing the complete EDA
